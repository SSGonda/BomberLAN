### How to Run

#### Server

To start the server,

Navigate to `haskell`

Run `cabal run haskell -- --host`

This should start the server in `127.0.0.1` in port `15000`

#### Client

To start a client,

Navigate to `haskell`

Run `cabal run`

This should run a client that connects to `127.0.0.1` in port `15000`
